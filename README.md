## 🛠 How it Works / 自动化原理

本项目通过 GitHub Actions 实现全自动同步，无需人工干预：

1. **监控**：每小时检查一次原仓库的最新 Release。
2. **校验**：对比 NPM 远程版本号与 GitHub 最新 Tag。
3. **分发**：若检测到新版本，脚本会自动下载 `.zip` 资产，重新打包并发布至 NPM。
